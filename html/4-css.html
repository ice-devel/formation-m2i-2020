Appliquer du style sur les éléments HTML

  • CSS via des feuilles de style externe : méthode à préférer (plus propre et réutilisable dans plusieurs pages
  • CSS en-ligne : directement dans les balises html

Cibler un élément HTML

  • par id : # (sélection d'un seul élément, un id doit être unique)
  • par classe : . (sélection d'un ou plusieurs éléments)
  • par tag (par nom de balise)

Priorité des sélecteurs

  • Les feuilles de style sont lues de haut en bas : les derniers styles prennent le dessus sur les styles précédents si les mêmes propriétés sont utilisées à nouveau
  • Il n'y a pas de priorité entre plusieurs blocs de style, si les sélecteurs utilisés sont les mêmes (le dernier prend le dessus)
  • En revanche, il y a des priorités entre plusieurs types de sélecteur différents. Ce n'est donc pas forcément le dernier style déclaré qui est appliqué : !important > style "en-ligne" > sélecteur ID > sélecteur classe > sélecteur tag
