Exercice : Créer en html 4 div avec chacun un id En jquery mettre une couleur de bordure différente pour chacun des divs Créer en html 3 boutons : ajouter - déplacer - remplacer - Quand vous cliquez sur le bouton "ajouter" : il faudra en jQuery ajouter un nouveau div avec une autre couleur de bordure et il faudra le placer après le troisième div - Quand vous cliquez sur le bouton déplacer, vous devez deplacer le premier div pour le mettre en dernière position - Quand vous cliquez sur le bouton remplacer, vous devez remplacer le 4éme div par un image dont vous aurez précisé la source
