Exercice : Jeu

Codez avec html/css/javascript/jquery le jeu où il faut cliquer sur une grille pour découvrir des images.
Vous devez faire une grille de 4 par 4, et dans chaque case il faut mettre une image.
Les image sont cachées, elles apparaissent quand on clique.
On joue en cliquant sur deux images
- on les fait apparaitre
- si elles sont identiques, on les laisse affichées
- si elles sont différentes, on les recache
- le jeu est terminé quand toutes les images sont affichées

Le code :
- créer le html : des div avec flex : 4 * 4
- jquery : créer une image dans chaque div, en sachant que chaque image doit exister
uniquement en double exemplaire - jquery détecter le clic sur les divs et afficher une image, puis une seconde
- et lancer une vérification quand deux images sont affichées (puis on les laisse affichées ou on les recache, c'est selon)
- compter le nombre d'essai effectués puis afficher à la fin de la partie combien d'essais ont été faits

Voir les fonctions :
- setTimeout()
- fadeIn() et fadeOut() : $(sel).fadeIn()
